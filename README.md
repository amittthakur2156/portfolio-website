# 🌐 Personal Portfolio Website

A modern and responsive **Personal Portfolio Website** built using **HTML, CSS, and JavaScript** to showcase my skills, projects, education, certifications, and contact information. The website is designed with a clean UI and smooth animations to provide a professional user experience.
